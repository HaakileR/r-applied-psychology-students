# Reproducibility note

These materials are organized as a small reproducible R project.

Recommended practice:

1. Open the `.Rproj` file.
2. Keep all file paths relative to the project root.
3. Run scripts in numerical order.
4. Save generated outputs in `outputs/tables/` and `outputs/figures/`.
5. Do not overwrite raw data.
6. Document all changes in `CHANGELOG.md`.

The synthetic data can be regenerated in future versions if needed, but the current release keeps a fixed dataset for stable teaching examples.
