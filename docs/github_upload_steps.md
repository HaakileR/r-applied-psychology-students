# GitHub Upload Steps

1. Open the repository on GitHub.
2. Click `uploading an existing file`.
3. Unzip the project ZIP file on your computer.
4. Open the unzipped folder.
5. Select all files and folders inside it.
6. Drag them into the GitHub upload window.
7. Commit message:

```text
Initial release of open R materials
```

8. Click `Commit changes`.

Important: upload the contents of the folder, not the ZIP file itself.
