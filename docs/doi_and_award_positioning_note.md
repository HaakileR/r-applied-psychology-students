# DOI and Academic Positioning Note

This repository can be positioned as an open educational and research training resource.

Possible contribution language:

> This open educational resource supports the development of data analysis skills among applied psychology students, with emphasis on reproducible research, R programming, and transparent reporting in psychological research. The material includes synthetic data, annotated R scripts, exercises, teaching notes, and reporting templates, and is publicly available through GitHub and archived with a DOI through Zenodo.

Possible evidence:

- public GitHub repository;
- Zenodo DOI;
- CITATION.cff file;
- license;
- README documentation;
- teaching use or workshop use;
- screenshots or confirmation of use in teaching/training.

Important note:

A DOI makes the material citable and easier to document, but the exact scoring in any award or evaluation depends on the official rules of that call.
