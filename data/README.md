# Data folder

This folder contains synthetic data for teaching and practice.

## Files

- `raw/student_psychology_synthetic_raw.csv`  
  Synthetic item-level dataset before scale scoring.

- `processed/student_psychology_synthetic_scored.csv`  
  Synthetic dataset with calculated scale scores.

- `data_dictionary.csv`  
  Variable names, descriptions, and value ranges.

## Important note

The dataset is fully synthetic. It does not contain real participant data and should not be interpreted as empirical evidence about real students.
