# Seven-Week Lesson Plan

## Week 1 — Introduction to R and RStudio

Topics:

- What is R?
- What is RStudio?
- R projects and folder structure
- Objects, vectors, functions
- Loading packages

Materials:

- `scripts/00_setup.R`
- `scripts/01_project_orientation.R`
- `exercises/exercise_01_intro_to_R.md`

## Week 2 — Importing and inspecting data

Topics:

- CSV files
- Variable names
- Rows and columns
- `glimpse()`, `summary()`, `head()`
- Basic frequency tables

Materials:

- `scripts/02_import_and_inspect_data.R`

## Week 3 — Data cleaning and missing values

Topics:

- Factor variables
- Numeric variables
- Missing values
- Complete cases
- Why raw data should not be overwritten

Materials:

- `scripts/03_cleaning_and_missing_values.R`
- `exercises/exercise_02_data_cleaning.md`

## Week 4 — Scale scoring and reliability

Topics:

- Likert-type items
- Reverse scoring
- Mean scale scores
- Cronbach's alpha
- Documentation of scoring decisions

Materials:

- `scripts/04_scale_scoring_and_reliability.R`

## Week 5 — Descriptives and visualization

Topics:

- Means and standard deviations
- Grouped summaries
- Histograms
- Boxplots
- Scatterplots
- Bar charts

Materials:

- `scripts/05_descriptive_statistics.R`
- `scripts/06_visualization_ggplot2.R`
- `exercises/exercise_03_descriptives_visualization.md`

## Week 6 — Group comparisons and correlations

Topics:

- Independent samples t-test
- ANOVA
- Correlation matrix
- Correlation tests
- Interpretation of p-values and effect direction

Materials:

- `scripts/07_group_comparisons.R`
- `scripts/08_correlation_analysis.R`
- `exercises/exercise_04_group_comparisons.md`

## Week 7 — Regression and reporting

Topics:

- Simple linear regression
- Multiple regression
- Interpreting coefficients
- Model fit
- Exporting tables and figures
- APA-style reporting examples

Materials:

- `scripts/09_regression_analysis.R`
- `scripts/10_export_results.R`
- `scripts/11_reporting_results_APA_style.R`
- `exercises/exercise_05_regression.md`
