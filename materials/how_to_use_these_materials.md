# How to Use These Materials

These materials are designed for introductory teaching and self-study in R-based data analysis for applied psychology.

## Recommended workflow

1. Open the `.Rproj` file in RStudio.
2. Run `scripts/00_setup.R`.
3. Open the dataset from `data/processed/`.
4. Work through scripts in numerical order.
5. After each script, complete the matching exercise.
6. Save your own outputs in the `outputs/` folder.

## Teaching format

The materials can be used in:

- one-day workshops;
- semester-based practical classes;
- independent student assignments;
- research methods and statistics courses;
- methodological training for psychology students.

## Suggested pace

For a short workshop:

- Session 1: RStudio, importing data, inspecting data
- Session 2: cleaning, missing values, scoring scales
- Session 3: descriptives, visualizations, t-tests
- Session 4: correlations, regression, reporting

For a semester course:

- Use the 7-week lesson plan in `lesson_plan_7_weeks.md`.

## Data note

All data are synthetic and used only for teaching.
